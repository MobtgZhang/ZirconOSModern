Original wallpaper designs for ZirconOS Modern theme.

| File | Resolution | Theme | Description |
|------|-----------|-------|-------------|

| `modern_purple.svg` | 1920x1080 | Purple | Solid purple accent background |
| `modern_green.svg` | 1920x1080 | Green | Solid green accent background |
| `modern_orange.svg` | 1920x1080 | Orange | Solid orange accent background |
| `modern_red.svg` | 1920x1080 | Red | Solid red accent background |
| `modern_dark.svg` | 1920x1080 | Dark | Dark gray background with minimal geometric pattern |

## Design Guidelines

- **Color palette:** Bold solid accent colors from the Metro palette
- **Style:** Flat, minimal, solid-color or subtle geometric patterns
- **No gradients:** Pure flat surfaces following Metro design language
- **Resolution:** 1920x1080 minimum, SVG format for scalability
- **Format:** SVG preferred for resolution independence
- All designs are original to the ZirconOS project
