## Design Language

All icons follow the ZirconOS Modern visual language:

- **Monochrome**: Single-color (white) outlines on colored tile backgrounds
- **Flat**: No gradients, no shadows, no 3D effects
- **Geometric**: Clean geometric shapes with uniform stroke width
- **Minimal**: Reduced detail, clear silhouettes

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| White | `#FFFFFF` | Icon outlines on dark/colored backgrounds |
| Black | `#000000` | Icon outlines on light backgrounds |
| Accent Blue | `#0078D7` | Accent highlights |

## Technical Details

- Format: SVG (Scalable Vector Graphics)
- ViewBox: `0 0 48 48` for all icons
- Stroke width: 2px uniform
- No external dependencies or linked resources
- Compatible with standard SVG renderers
